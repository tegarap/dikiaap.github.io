Membuat User Di Mysql Dan Hak Akses User Mysql

Jun 19, 2015 • dikiaap

MySQL adalah sebuah perangkat lunak sistem manajemen basis data SQL (bahasa Inggris: database management system) atau DBMS yang multithread, multi-user, dengan sekitar 6 juta instalasi di seluruh dunia. MySQL AB membuat MySQL tersedia sebagai perangkat lunak gratis dibawah lisensi GNU General Public License (GPL), tetapi mereka juga menjual dibawah lisensi komersial untuk kasus-kasus dimana penggunaannya tidak cocok dengan penggunaan GPL.

Tidak sama dengan proyek-proyek seperti Apache, dimana perangkat lunak dikembangkan oleh komunitas umum, dan hak cipta untuk kode sumber dimiliki oleh penulisnya masing-masing, MySQL dimiliki dan disponsori oleh sebuah perusahaan komersial Swedia MySQL AB, dimana memegang hak cipta hampir atas semua kode sumbernya. Kedua orang Swedia dan satu orang Finlandia yang mendirikan MySQL AB adalah: David Axmark, Allan Larsson, dan Michael “Monty” Widenius.

Oke langsung aja ya. Pembuatan user di mysql digunakan untuk melindungi ancaman serangan dari luar (dalam keterangan ini bukan user di database khusus untuk sebuah web), karena sebenarnya saat kita login di server msyql adalah login sebagai root (administrator) yang bisa mempunyai hak akses penuh ke semua data. Maka kita harus membuat user baru untuk mencegah ancaman sewaktu2. Dalam pengaksesan kali ini, saya menggunakan jaringan lokal dan dibutuhkan 2 komputer / lebih untuk bisa saling mengakses / login di mysql server. Cara ini juga bisa diterapkan di sebuah paket hosting.

Pertama, kita jalankan dulu service mysql di xampp atau mysql-server. Jika sudah berhasil berjalan, kita login sebagai root dulu.

root@pickgoo:~# mysql -h 127.0.0.1 -u root -P 3306

Note : Dalam hal ini saya sudah membuat database beserta table nya.

Buatlah Lan / Wifi untuk terkoneksi ke komputer lain.

Ip User 1 (server) = 200.200.7.27

Ip User 2 (client) = 200.200.7.98

Settinglah ip nya sesuai dengan keinginan kalian, yap saya rasa kalian sudah mensetting ip nya.

Untuk mengecek kita terkoneksi atau tidaknya ke user 2 kita harus mengirim request ke komputer user 2

Sekarang saatnya untuk membuat user, ini codenya

mysql> create user 'diki'@'200.200.7.98' identified by 'admin99';

Keterangan :

  • ip 200.200.7.98 adalah ip komputer lain, yang digunakan untuk mengakses (bkn ip komputer sendiri)

  • admin99 adalah password user tersebut

Selanjutnya, kita beri hak akses user tersebut.

Saya akan memberi hak akses kepada user diki hanya untuk melihat dan menambahkan data ke TBL_PETUGAS

mysql> grant select, insert on accsess.TBL_PETUGAS to 'diki'@'200.200.7.98';

Membuat dan mensetting user sudah selesai, saatnya mengetest berjalan atau tidak user tersebut.

Kita login dulu di komputer user 2 (200.200.7.98)

mysql -h 200.200.7.27 -u diki -p

Keterangan :

  • ip 200.200.7.27 adalah ip user 1 (server)

  • diatas kita sudah dapat mengakses database “accsess” Selanjutya kita lihat, tabel apa saja yang bisa di akses.

mysql> use accsess; //(untuk menggunakan database accsess)

mysql> show tables; //(untuk melihat table)

Oke, kita coba menambahkan field ke tabel TBL_PETUGAS

mysql> insert into TBL_PETUGAS values ("1", "Petugas ");

Untuk melihat field yang kita tambahkan tadi kita bisa menulis perintah

mysql> select * from tbl_petugas;

Kita coba delete data yang kita tambahkan tadi.

mysql> delete from tbl_petugas where NIP=’1’;

Ternyata tidak bisa menghapus, karena hak aksesnya hanya melihat(select) dan insert saja.
