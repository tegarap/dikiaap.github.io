Membuat Host Only Network

14 Jun 2015 • dikiaap •

Berawal dari pc pas-pas an, alias hanya ada 1 di rumah. Saya berpikiran untuk membangun sebuah lokal network di rumah, yaitu dengan 1 pc dan bantuan 2 OS, yakni OS host dan OS virtual machine di virtual box. Yah, Apa daya cuma punya 1 pc. Tapi tidak apa2. Ini cukup membantu bagi yang ingin ngetest system yang butuh 2 OS / lebih. Nama Koneksi Jaringan ini adalah Hostonly Network Pertama Buka Virtual Box.

Pertama, Setting Network di vbox File > Preferences

Pilih Network > Lalu Pilih Tab Host-only Network > Tombol Add

Klik gambar obeng Untuk mensetting Ip Address Beserta Subnetmasknya

NOTE:

  • Perlu diingat IP harus 192.168.56.1 karena itu defaultnya vbox, kalau diganti mungkin tidak akan terdeteksi di jaringan lokalnya

  • DHCP Server tidak usah dicentang, karena kita hanya menggunakan ip address static saja

Sekarang pilih OS anda

Pilih Network > Tab Adapter 1

Lalu setting IP nya diwindows (virtual machine), konfigurasi diwindows ini dideteksi sebagai LAN

Sekarang konfigurasi jaringan di komputer kita, konfigurasi di host ini harus memakai eth0

Edit juga network interfaces nya

root@pickgoo:~# nano /etc/network/interfaces

auto eth0

iface eth0 inet static

address 192.168.56.55

netmask 255.255.255.0

network 192.168.56.0

broadcast 192.168.56.254

post-up route add default gw 192.168.56.1 metric 2

pre-down route del default gw 192.168.56.1

gateway 192.168.56.1

Note : Firewall di OS host dan OS virtual machine harus di disable, Kalau tidak mungkin koneksi nya akan di block oleh firewall tadi
