Install Laravel 5 Di Ubuntu

Jun 17, 2015 â€¢ dikiaap

Installasi

Rekomendasi System

  • PHP >= 5.5.9

  • OpenSSL PHP Extension

  • Mbstring PHP Extension

  • Tokenizer PHP Extension

Ada 3 Macam Installasi Laravel, yaitu :

  • Install Laravel Lewat Composer

  • Install Laravel Lewat Composer Create-Project

  • Install Laravel Lewat Source Code

Silahkan pilih untuk kemudahan dan kegunaan yang cocok buat anda

Install Laravel Lewat Composer

sudo apt-get install curl

curl -sS https://getcomposer.org/installer | php

sudo mv composer.phar /usr/local/bin/composer

composer global require "laravel/installer=~1.1"

Executable laravel berada di ~/.composer/vendor/bin/laravel

Agar lebih mudah dijalankan, kita buat alias untuk laravel

nano ~/.bashrc

Sisipkan kode ini alias laravel='/home/username/.composer/vendor/bin/laravel'

Jangan lupa simpan

Saya memakai nginx, File local web server saya di /var/www/

cd /var/www

laravel new blog blog disini adalah folder project kamu.

Install Laravel Lewat Composer Create-Project

Kamu juga dapat install laravel menggunakan Composer create-project lewat terminal

composer create-project laravel/laravel blog --prefer-dist

Install Laravel Lewat Source Code

Cara ini banyak digunakan oleh programmer, karena kemudahannya

Source : Github Laravel

Tapi saya rekomendasikan buat yang mempunyai koneksi internet stabil untuk mendownload secara online, karena pasti akan mendapatkan update terbaru library library dari laravel.
