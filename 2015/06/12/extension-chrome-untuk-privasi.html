Extension Chrome Untuk Privasi

Jun 12, 2015 • dikiaap

Helo!!!!!! Yah. saya akan share beberapa extension yg menurut saya penting untuk privasi

Fungsi utamanya untuk mengurangi website mendapatkan informasi & identitas anda , dll Pertama,

Adblock Plus { }

Adblock Plus (ABP) adalah ekstensi penyaringan konten dan pemblokiran sumber terbuka, AdBlock memungkinkan pengguna mencegah munculnya elemen web tertentu seperti iklan yg tidak diinginkan dan dianggap sbg ancaman

Don’t Track Me Google { }

Ekstensi ini akan menyingkirkan url filter google asli, menyembunyikan informasi. sehingga memungkinkan anda langsung ke situs tujuan, dan google tidak akan merekam aktifitas kunjungan situs anda

Awalnya diterbitkan di http://userscripts.org/scripts/show/121923 (lebih 36K pengguna).

Avast Online Security { }

Mempunyai fungsi yg sama dgn adblock, alangkah baik jika anda menambahkan extension ini, karena ini dapat menambah keamanan browser anda.

  • Mengumpulkan data situs phishing dan memperingatkan Anda jika Anda mengunjungi salah satu situs phising tersebut.

  • Memberikan Anda akses ke sistem reputasi situs, didukung oleh komunitas pengguna lebih dari 220M.

  • Menunjukkan rating situs sebelah link dalam hasil pencarian Anda, berbahaya atau tidak.

  • Memperingatkan Anda ketika Anda mengunjungi situs dengan reputasi buruk.

  • Memungkinkan Anda untuk menilai sebuah situs dengan acungan jempol sederhana atau jempol ke bawah, sehingga meningkatkan sistem dan membantu pengguna lain.

  • Secara otomatis Anda akan diarahkan ke situs yang sah ketika Anda salah ketik URL (misalnya, llego.com menjadi lego.com).

  • Blok iklan dan analisis situs pengganggu dan membuat Anda lepas dari pengintaian secara online.

Flash Control { }

Manfaat :

  • Penghematan CPU dan penggunaan daya

  • Penjelajahan lebih tenang

Memiliki kontrol:

  • Auto-play video

  • Pemain Musik

  • Permainan

  • Dan beberapa iklan banner

Ghostery { }

Ghostery mempunyai fungsi seperti adblock dan aos.

Ghostery dibuat dan dikelola untuk pengguna yang peduli tentang privasi online, dan direkayasa dengan privasi sebagai tujuan utama. Penggunaan Ghostery itu anonim. Tidak ada pendaftaran atau sign-up yang diperlukan. Ghostery tidak menempatkan cookies dalam browser Anda. Ghostery akan menolak iklan, tracking, web analytics, widget, adversiting, beacons.

Https Everywhere { }

Jika anda menggunakan extensions ini, Secara otomatis anda akan di pandu dari “http” untuk menuju ke “https”. Ini akan melindungi Anda terhadap bentuk pengawasan dan pembajakan akun, dan beberapa bentuk penyensoran.

ScriptSafe { }

Sebuah ekstensi sederhana yang mempunyai fungsi sama seperti NoScript

Web Of Trust { }

WOT adalah extension untuk melihat sebuah reputasi situs dan layanan yang membantu Anda untuk membuat keputusan tentang apakah akan mempercayai sebuah situs web tersebut atau tidak. Ketika Anda melakukan pencarian, belanja atau berselancar online.

Zen Mate Security & Privacy VPN { }

Sperti Anonymox. ZenMate Mengenkripsi dan mengamankan semua lalu lintas browser Anda! ZenMate Security & Privacy VPN memberikan perlindungan bagi Anda.
