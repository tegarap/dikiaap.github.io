Pemrogramman Bash Sederhana

20 Jun 2015 • dikiaap •

Bourne-Again shell Dari Wikipedia bahasa Indonesia, ensiklopedia bebas. Bourne-Again shell lebih sering disebut dengan bash. Bash ini salah program dalam GNU Project yang menjadi shell yang paling banyak digunakan pada lingkungan *NIX. Bourne-Again shell merupakan pengembangan dari Bourne shell. Namanya diambil dari penciptanya, yakni Stephen Bourne.

Bash sendiri kemudian dikembangkan Brian Fox. Saat ini, bash menjadi script’ primer pada sistem Linux dan telah disertakan pula di Mac OS X Tiger. Selain Bash, shell yang ada ialah C-shell dan Korn Shell. Perbedaan ketiganya terlihat pada format penyimpananya, yaitu Bash (.sh), C-shell (.csh), dan Korn Shell (.ksh).

Contoh ini hanya dasarnya saja ya. Buka tools editing kalian, ketik dan simpan dengan extension .sh. Untuk jalaninnya dengan cara bash namafile.sh atau ./namafile.sh

Nih contohnya

File name : break.sh

#!/bin/bash
# file : break.sh
total=0
jumlah=0
echo "Akhiri dengan memasukan angka 0 atau negatif"
while true
do
echo "Nilai : "
read nilai
if [  $nilai -le 0 ]
then
      break
fi
jumlah=`expr $jumlah + 1`
total=`expr $total + $nilai`
done
rata=`expr $total / $jumlah`
echo "Rata-rata $rata"

File name: cobawhile.sh

#!/bin/bash
# file : cobawhile.sh
i=1
while [  "$i" -lt 7 ]
do
echo $i
i=`expr $i + 1`
done

File name: elif.sh

#!/bin/bash
# file : elif.sh
OPTIONS="Hello Quit"
select opt in $OPTIONS; do
if [ "$opt" = "Quit" ]; then
echo done
exit
elif [ "$opt" = "Hello" ]; then
echo Hello world
else
clear
echo bad option
fi
done

File name: hari.sh

#!/bin/bash
# file : hari.sh
kodehari=`date +%w `
case $kodehari in
0) echo "Hari Minggu" ;;
1) echo "Hari Senin" ;;
2) echo "Hari Selasa" ;;
3) echo "Hari Rabu" ;;
4) echo "Hari Kamis" ;;
5) echo "Hari Jumat" ;;
6) echo "Hari Sabtu" ;;
esac

File name: ucapan.sh

#!/bin/bash
# file : ucapan.sh
date
jam=`date +%H`
if [ $jam -lt 10 ]
then
    echo "Selamat pagi"
else
    if [ $jam -lt 15 ]
    then
        echo "Selamat siang"
    else
        if [ $jam -lt 17 ]
        then
            echo "Selamat sore"
        else
            echo "Selamat lembur"
        fi
    fi
fi
